DummyJSON API Integration

Powered by DummyJSON

AZIDMart menggunakan DummyJSON API sebagai sumber data utama. DummyJSON adalah REST API gratis yang menyediakan data dummy realistis untuk pengembangan dan testing.

Endpoint Method Deskripsi Status
/products GET Ambil semua produk Active
/products/{id} GET Detail produk by ID Active
/products/categories GET Semua kategori Active
/products/category/{name} GET Produk by kategori Active
/products/search?q={query} GET Search produk Active
